# caddy
一个开箱即用、集成常用扩展的 Caddy 容器镜像

# 拉取镜像
### 国内（腾讯云广州）
```bash
docker pull ccr.ccs.tencentyun.com/xiaoqidun/caddy:latest
```
### 海外（腾讯云香港）
```bash
docker pull hkccr.ccs.tencentyun.com/xiaoqidun/caddy:latest
```
